Esse é um projeto feito para uma empresa fabricande de alimentos naturais para cães e gatos. a Wow pet tem como objetivo, uma alimentação saudável para os pets.

O projeto será desenvolvido em em Wordpress, e Woocommerce com Elementor. o grande desafio desse projeto são os planos de assinatura personalizados. Onde a pessoa preeenche os dados completos dos pets, para assim identificar o tipo de alimento perfeito para cada um deles.
